Retningslinjer for informasjonskapsler

Hvordan DeliGo bruker informasjonskapsler og lignende teknologier på www.deligo.pt og våre mobilapper.

1. Om denne policyen

Disse retningslinjene for informasjonskapsler forklarer hva informasjonskapsler er, hvordan vi bruker dem og hvilke valg du har. De gjelder for nettstedet www.deligo.pt og relaterte tjenester som drives av PIXELMIRACLE, LDA (DeliGo)for mer informasjon om hvordan vi behandler personopplysninger, vennligst les vår Retningslinjer for personvern.

2. Hva er informasjonskapsler?

Informasjonskapsler er små tekstfiler som lagres på enheten din når du besøker et nettsted. De hjelper nettstedet med å huske dine handlinger og preferanser (for eksempel innlogging, språk og region) over en viss tidsperiode. Vi kan også bruke lignende teknologier som SDK-er i mobilappene våre og lokal lagring i nettleseren din.

3. Typer informasjonskapsler vi bruker

  • Essensiell (strengt nødvendig) - Nødvendig for at nettstedet skal fungere (f.eks. sikkerhet, økt, lastbalansering). Disse kan ikke slås av.
  • Ytelse og analyse - Hjelpe oss med å måle trafikk, bruk og forbedre funksjoner (aggregert statistikk).
  • Funksjonell - Husk valg (språk, tema) og forbedre opplevelsen din.
  • Markedsføring - Brukes til å levere og måle annonser som er relevante for deg på tvers av nettsteder og apper.

4. Informasjonskapsler vi kan sette

Merk: De nøyaktige informasjonskapslene og varighetene kan variere over tid, ettersom vi forbedrer tjenestene våre eller bytter leverandør. Vi oppdaterer denne siden når det skjer vesentlige endringer.

5. Administrere preferansene dine

Du kan velge hvilke valgfrie informasjonskapsler du vil godta, ved å bruke verktøyet for innstillinger for informasjonskapsler. Du kan også blokkere eller slette informasjonskapsler via innstillingene i nettleseren eller på enheten din. Blokkering av viktige informasjonskapsler kan imidlertid påvirke nettstedets funksjonalitet.

Åpne innstillinger for informasjonskapsler

  • Nettleserkontroller: Se hjelpesidene for Chrome, Firefox, Safari eller Edge for instruksjoner om hvordan du sletter eller blokkerer informasjonskapsler.
  • Mobilapper: Bruk enhetsinnstillinger (f.eks. "Tilbakestill annonserings-ID" eller "Begrens annonsesporing").
  • Fravalg av analyser: Noen analyseleverandører tilbyr egne opt-out-mekanismer.

6. Juridisk grunnlag

Der loven krever det, ber vi om ditt samtykke for ikke-essensielle informasjonskapsler (f.eks. analyse og markedsføring). Viktige informasjonskapsler er basert på vår legitime interesse i å levere sikre og pålitelige tjenester. Du kan når som helst trekke tilbake samtykket ditt via preferanseverktøyet.

7. Oppdateringer av disse retningslinjene

Vi kan oppdatere disse retningslinjene for informasjonskapsler for å gjenspeile endringer i teknologi, lovgivning eller tjenestene våre. Eventuelle endringer vil bli lagt ut på denne siden med en ny "Sist oppdatert"-dato.
